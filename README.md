# Agence Voyage - VoyageDream

## 🌍 Projet de Site Vitrine pour Agence de Voyage

Projet réalisé dans le cadre de l'atelier créatif - Samba B. KANE

### 📋 Description du Projet

Ce projet consiste à créer un site vitrine fonctionnel pour une agence de voyage nommée **VoyageDream**. Le site présente les services de l'agence, les destinations populaires et permet aux clients de prendre contact.

- ✅ Organiser un travail en équipe de manière autonome et structurée
- ✅ Planifier un projet en utilisant une méthodologie agile
- ✅ Collaborer efficacement via GitHub
- ✅ Produire un premier livrable fonctionnel et exploitable
- ✅ Analyser et améliorer notre manière de travailler

### 🚀 Fonctionnalités Implémentées

#### 📱 Design Responsive
- Adaptation parfaite sur mobile, tablette et desktop
- Menu hamburger pour les écrans mobiles
- Grilles flexibles et media queries

#### 🎨 Interface Moderne
- Design moderne et épuré avec la police Poppins
- Animations CSS et JavaScript fluides
- Effets de survol et transitions élégantes
- Header fixe avec effet de transparence

#### ⚡ Interactivité
- Navigation smooth scroll
- Animations au défilement (Intersection Observer)
- Formulaire de contact fonctionnel avec validation
- Animation des statistiques
- Effet parallaxe sur la hero section

#### 📄 Pages du Site
1. **Page d'accueil** : Présentation complète de l'agence
2. **Destinations** : Catalogue des voyages avec filtres
3. **À Propos** : Histoire et valeurs de l'agence
4. **Contact** : Formulaire de contact et informations
5. **Mentions Légales** : Conformité légale complète

### 🛠️ Technologies Utilisées

- **HTML5** : Structure sémantique et accessible
- **CSS3** : Design moderne avec Grid et Flexbox
- **JavaScript Vanilla** : Interactivité sans framework
- **Google Fonts** : Typographie Poppins
- **GitHub** : Versioning et collaboration

### 🚀 Comment Lancer le Projet

1. **Cloner le repository** :
   ```bash
   git clone https://github.com/Jean207col/Agence_Voyage.git
   cd Agence_Voyage
   ```

2. **Ouvrir le site** :
   - Ouvrir `index.html` dans votre navigateur
   - Ou utiliser un serveur local pour une meilleure expérience

3. **Ajouter des images** :
   - Placer vos images dans le dossier `assets/images/`
   - Noms recommandés : `hero-bg.jpg`, `paris.jpg`, `bali.jpg`, `newyork.jpg`, `tokyo.jpg`

### 🔧 Personnalisation

Pour adapter le site à vos besoins :

1. **Modifier les couleurs** : Dans `css/style.css`, cherchez les variables de couleur
2. **Changer les textes** : Éditez directement les fichiers HTML
3. **Ajouter des destinations** : Copiez les cartes existantes dans `index.html`
4. **Personnaliser le formulaire** : Modifiez `js/script.js` pour changer le comportement
